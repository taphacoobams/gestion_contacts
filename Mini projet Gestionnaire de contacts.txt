Mini Projet Gestion de contacts
	Fonctionnalites:
		Lister les contacts
		ID Nom, Prenom, Adresse, Email, Telepehone, Description (TEXTAREA)
		Ajouter un contact 
		Modifier un contact
		Supprimer un contact 
		Afficher la page de detail d'un contact 

	Front-End: Angular
	
	BACK-END: API REST (WEB Service) : Ressources 
		GET contacts https://localhost:8080/contacts
		GET contacts https://localhost:8080/contacts/{idContact}	 
		DELETE contacts https://localhost:8080/contacts
		POST contacts https://localhost:8080/contacts
			BODY 
			{
				"nom": "DIAW",
				"prenom": "AMADOU",

				"telephone": "+222 787 55 21",
				"description": "Je suis un ingenieur en Informatique avec plus de 4 ans d'exprience",
			}
	
		PUT contacts https://localhost:8080/contacts/{idContact}
			BODY 
			{

		Utiliser POSTMAN pour tester votre API

			JAVA: SPRING REST
